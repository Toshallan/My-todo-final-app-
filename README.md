# TodoList App - Project

Une application de gestion de tâches complète développée en JavaScript Vanilla. Conçue comme projet académique pour l'année universitaire 2025/2026.

##  Fonctionnalités

* **Système d'authentification simulé** : Connexion par nom d'utilisateur, création automatique de l'espace personnel.
* **CRUD complet** : Ajouter, afficher, marquer comme terminé et supprimer des tâches.
* **Persistance des données** : Sauvegarde intégrale dans le `localStorage` du navigateur (Sessions et Tâches).
* **Interface dynamique** : 
    * Statistiques en temps réel (tâches totales / terminées).
    * Animations fluides (ajout et suppression).
    * Mode sombre/clair persistant.

##  Technologies utilisées

* HTML5
* CSS3 (Variables, Animations)
* JavaScript (DOM, ES6, LocalStorage)

##  Déploiement via GitHub Pages

1. Créez un nouveau dépôt sur GitHub.
2. Uploadez les fichiers (`index.html`, `style.css`, `script.js`, `README.md`).
3. Allez dans **Settings > Pages**.
4. Sous "Source", sélectionnez la branche `main` et sauvegardez.
5. Votre site sera accessible sur : `https://toshallan.github.io/My-todo-final-app-/`
